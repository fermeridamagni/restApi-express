# RestAPI in ExpressJS

This project is a Example for REST-API in Express, using MYSQL as Database.


**Packages in use:**

* express,
* express-mysql-session,
* express-session,
* express-validator,
* mysql2,
* ejs,
* nodemon,
* morgan,
* dotenv,
* uuid
